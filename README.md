College Canteen Mobile Application

Overview
This project is an Android-based College Canteen Management System designed to streamline food ordering and reduce congestion during peak hours. It enables students to pre-order food, track order status, and make payments digitally.

Objective
* Reduce crowd and waiting time in canteens
* Enable digital ordering and payment
* Improve efficiency for students and staff

Modules
Student Module
  * Registration & Login
  * View menu and add items to cart
  * Place orders
  * Select payment method (Cash / E-wallet)
  * Track order status and history 
Admin Module
  * Manage menu (add/update/delete items)
  * View all orders
  * Update delivery status
  * Manage student wallet balance 
Kitchen Module
  * View pending orders
  * Update order status (Preparing / Ready) 

Database Design
  * Student Table
  * Menu Table
  * Wallet Table
  * Orders Table 

Tech Stack
  * Java (Android)
  * XML (UI Design)
  * SQLite (Database)
  * Android Studio

Features
  * Digital food ordering system
  * Real-time order tracking
  * E-wallet integration
  * Reduced manual errors and paperwork
  * Efficient order management system

Results
  * Successfully reduced congestion during peak hours
  * Improved order accuracy and tracking
  * Enabled faster service and digital transactions 

How to Run
1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run on emulator or Android device

Future Enhancements
